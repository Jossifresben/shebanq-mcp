Ask the Hebrew Bible

A plain-language question becomes a real MQL query: shown, editable, and run live.

An AI drafts an Emdros MQL query over the BHSA database (the data behind SHEBANQ). The query is validated, then run on the shebanq MCP server. It is always shown: read it, edit it, re-run it. AI as a way in, not a way around.

This demo runs the shebanq MCP server on Render's free tier, so the first request after an idle spell can take up to a minute while the server wakes; later queries are fast. It is read-only: you can search the data, not change it. The AI translation runs on a small monthly budget, so it may occasionally pause, in which case you can still write and run MQL by hand.
