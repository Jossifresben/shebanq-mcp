Query the BHSA in plain language

Text-Fabric ⇄ MQL converter · About

A plain-language question becomes an MQL query: shown, editable, and run live.

Anthropic's Claude (Opus 4.8) drafts an Emdros MQL query over the BHSA, the ETCBC's linguistic database of the Hebrew Bible that powers SHEBANQ. The query is validated, then you can run the query through the shebanq MCP server.

The AI translation runs on a small monthly budget, so it may occasionally pause.
